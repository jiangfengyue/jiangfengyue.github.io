读《华为三十年》有感

发表于 2017-03-29 | 分类于 阅想

​ 华为，作为中国最神奇的民营企业，三十年的成长过程，为无数人津津乐道。无论是科技实力、市场营销手段、人才战略还是资本运作模式，华为有太多值得学习和探索的地方。今天我想就企业管理这一方面，谈谈对《华为三十年》一书的理解。

​ 华为是任正非一手创办，并在其规划和指引下，茁壮成长起来的。可以说，华为企业管理理念的变化与任正非管理思想的发展密切相关。华为经历了三个发展时期，分别对应着任正非三个典型的管理理念。在华为初创时期，企业小，管理难度低，对市场敏感度高，主要依靠领导者的个人能力。这个时期，任正非以毛泽东的游击战、“农村包围城市”等思想为主，以狼性的作风开疆扩土，在残酷的市场竞争中代领华为冲锋突围，建立基业。进入平稳发展期后，华为成为中型企业，管理难度提高，此时偏向于企业自身的优化和重组。任正非学习了郭士纳对IBM管理改革的成功经验，将IBM的管理制度移植到华为中，克服官僚作风，优化内部结构，使华为继续做大做强。在快速发展期，华为的国际化进程加快，在多个国家开拓了市场，建立了研发和生产机构，管理难度进一步加大，相应的管理体系也需要变革。任正非又向稻盛和夫取经，以阿米巴模式再造华为的管理架构，力图保持华为的竞争优势。任正非通过对三类管理思想的学习和改造，形成了华为独特的企业文化和管理理念，充分地调动了华为人的积极性和创造性，真正激活了企业的生命力，使华为能够适应市场的变化和企业的成长要求。

​ 每一个企业在市场上站稳脚跟后，都需要做大做强。而成为大企业之后，不可避免的，一些大企业的内部问题就会随之产生。其中最为突出也是最为根源的就是官僚主义。为了进一步细化工作，强化管理，大企业往往部门众多，决策链条和业务流程加长，部门之间的协调工作增加，员工紧迫感下降，形成了部门墙，造成决策层不熟悉市场的变化、管理层互相推诿拖延、执行层不能够快速响应市场需求及提供有效服务，导致企业上下脱节、左右脱钩，整体降低效率和收益。这是不可避免的事情，任何大企业都会存在这样的问题，不论是华为还是IBM。关键在于领导者能不能意识到这一点，普通员工有没有这样的觉悟。任正非在参观浴火重生的IBM后，意识到壮大起来的华为也会存在同样的问题。因此他向郭示纳取经，重金聘请IBM的顾问对华为的管理体系进行再造，从人力资源、财务体系、研发系统等多个方面进行深刻变革。然而变革也不是一蹴而就的。任正非充分认识到一个新制度的建立是一个过程，是一个需要被企业同化和吸收的过程。因此他采用“先僵化后优化再固化”的策略，逐步对各部门进行改造，实现了中央集权到分权制衡的变化，提高了基层的决策权力及责任，也就激发了更广大员工参与企业建设的动力，进一步将员工和企业绑定成一个整体。

​ 面对外部市场迅猛变化，如果企业不能快速的做出反应，就会像战场上不能快速反应的部队一样，被对手给歼灭。柯达、诺基亚、阿尔卡特等都是前车之鉴。因此，面对全球化的市场，华为采取阿米巴的经营模式，不断变革组织结构，化小利润中心，缩短决策链，加快决策速度。通过建立独立核算的部门和分公司，一方面能够简政放权，激活部门的活力和效能，另一方面能够发挥“以小博大”的竞争优势。每个分公司相当于初创时期的华为，充分发扬了华为的“狼性”传统，在市场上冲锋陷阵，同时母公司为其提供全球化平台的支持，整个华为进化成了有众多触须的海葵，总能够在意想不到的地方给对手以猝不及防的一击。

​ 可以看到，在华为的不同发展时期，任正非没有固执坚持一种管理模式，而是面对不同的市场和企业发展特点，思考其中的规律，不断引进国内外的先进管理经验，融合中国传统文化的精华，再通过自创思维的吸收和生化，形成了华为开放、包容、艰苦奋斗、实事求是的企业文化，从而保证了华为不断从一个胜利走向另一个胜利。
